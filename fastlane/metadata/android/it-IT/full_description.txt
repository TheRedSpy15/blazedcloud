Blazed Cloud è un servizio di backup cloud minimalista che ama il software open source.

- Tutti i file sono crittografati con AES-256
- L'interfaccia utente è semplice e non ingombra
- L'app stessa è leggera e non occuperà molto spazio
- 15 Gigabyte di livello gratuito solo creando un account
- Facile condivisione di file tramite URL con scadenze temporali
- Blocco dell'app tramite biometria
- Eliminazione opzionale dell'account se smetti di utilizzare il nostro servizio dopo 90 giorni.
