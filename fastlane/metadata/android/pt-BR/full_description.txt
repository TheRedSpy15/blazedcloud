Blazed Cloud é um serviço de backup na nuvem minimalista que ama software de código aberto.

- Todos os arquivos são criptografados com AES-256
- A interface é simples e não é confusa
- O próprio aplicativo é leve e não ocupa muito espaço
- 15 Gigabytes gratuitos apenas por criar uma conta
- Compartilhamento fácil de arquivos por URLs com expiração programada
- Bloqueio biométrico do aplicativo
- Exclusão opcional da conta se você parar de usar nosso serviço após 90 dias.
